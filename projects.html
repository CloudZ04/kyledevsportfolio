Action Against Stalking Website

Full Website Redesign & Development

2024-2025

  • A WordPress Elementor website that was re-designed for a modern look and feel.
  • Built using WordPress and Elementor, with some custom CSS & JavaScript.
  • Responsive design for both desktop and mobile devices.

While in college & university I had the opportunity to work on a client project for Action Against Stalking (AAS), where I contributed to design, development, and content structure with another student.

The CEO (our client) felt the original website was outdated, and was in need of updating. We decided the colour scheme to match the brand theme of the charity based on their logo, and created a new website that was more modern and user-friendly, while portraying the charity's values and mission.

I was also responsible for the design of a brand new logo for the 'Centre for Action Against Stalking' which I designed using Adobe Illustrator, taking inspiration from the charity's main logo, and values.

GameTracker

Final Grade Project for NCL

2025

  • Built a personal game tracking site
  • Technologies: HTML, CSS, JavaScript, PHP & MySQL
  • Features: API integration, manual database editing, friends and messaging system

GameTracker.gg was just a personal project that I created to track upcoming games that had already, or been scheduled to release within the 2025-2026 period. Although now I have re-developed the site to hold many more games going back as far as 30 years ago as I wanted to use the site as a personal collection.

I am a gamer myself. I love story games, I love the competitiveness some games bring (such as VALORANT), and it's just an escape for me. So I decided to create a site that would benefit myself, but thought of other gamers as they would be the target audience. Thankfully it paid off and this project got me an A for my HND!

The site uses API integration and a PHP and MySQL based back-end for manual database editing. This includes fetching data, searching, manual insertion, deletion, and editing of data. The site also has a friends and messaging system now as I wanted to try build something new. As of now, the site is not live, and I'm unsure if it ever will be as a site like this requires a lot of work and maintenance, especially with new games being released every day. Maybe in the future.
